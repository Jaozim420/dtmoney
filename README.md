# dtmoney

 # ⚙️ Languages:
- ReactJS
- TypeScript

# 📚 Frameworks:
- [miragejs](https://miragejs.com/) 
- [styled-components](https://styled-components.com/)
- [axios](https://github.com/axios/axios)

# Installing:
```bash
$ git clone git@github.com:Jaozim420/dtmoney.git
```

```bash
# Install Dependencies
$ yarn
# Run Aplication
$ yarn start
# Go to
$ http://localhost:3000/
```
